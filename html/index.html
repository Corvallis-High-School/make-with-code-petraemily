A history lesson of HTML that I made for this class.

Tim Berners-Lee Development Versions
Tim Berners-Lee is known as the inventor of HTML, the World Wide Web, and the URL system. He first worked for a telecommunications company, and eventually started working for CERN, proposing a project based on hypertext. That project later became ENQUIRE, the predecessor to the World Wide Web.
The development for HTML began in 1989, and the first edition of HTML was made publically availible in 1991. After the draft for the prototype expired in 1994, Berners-Lee and a group of others created HTML 2.0 in 1995, with further development from then to now.
After HTML and HTML 2.0, there were several other versions made, improving upon the last. HTML 3.0 was published in 1997, reccomended by the W3C. HTML 4.0 was published in 1998, and subsequently updated and reissued 1999 and 2000. HTML 5.0 was published in 2014 as a W3C reccomendation, and retired, along with all other versions, in 2018. The currect specification is HTML living standard, also called HTML5, confusingly.
